# GLUE Benchmark

The General Language Understanding Evaluation (GLUE) benchmark is a collection of nine NLU tasks (CoLA, SST-2, MRPC, STS-B, QQP, MNLI, QNLI, RTE, WNLI) used to evaluate language models.

Key relevance to this knowledge base: transformers — especially BERT and successors — achieved state-of-the-art on GLUE, demonstrating the power of [[self-attention]] for language understanding. The benchmark is a standard yardstick for comparing [[transformer-architecture]] variants.
