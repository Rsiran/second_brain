# Transformer Architecture Diagram

A high-level diagram of the transformer architecture showing the encoder-decoder structure. The encoder (left) processes the input sequence through stacked layers of [[self-attention]] and feed-forward blocks. The decoder (right) generates the output sequence with masked self-attention and encoder-decoder attention.

Illustrates the architecture described in [[attention-is-all-you-need]] and [[the-illustrated-transformer]].
